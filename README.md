# m5stack_operation_lamp

## Setting
1. rename src/wifi_pass_template.h to src/wifi_pass.h
2. add wifi-ssid, wifi-password, slack-webhook-url to src/wifi_pass.h
3. edit message for slack in src/wifi_pass.h
